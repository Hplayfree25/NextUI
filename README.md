<h3>Overview</h3>
<p>
  <b>Vantix UI</b> is an lightweight UI Library tailored for Roblox developers and scripters. Designed with a seamless aesthetic experience in mind, Vantix replaces standard interfaces with high-quality, modern, and fluid mechanics
</p>
<p>
  Featuring a <b>modern rounded UI</b>, intuitive tab-section layouts, and robust customizability
</p>

<h3>Project Status & Disclaimer</h3>
<blockquote>
  <b>Note:</b> This repository is currently a <strong>Side Project</strong>. Updates, patches, and feature additions are strictly on a rolling, as-time-permits basis. It is not currently under constant maintenance, so users should expect potential periods of inactivity
</blockquote>

<h3>Build & Integration</h3>
<p>
  Vantix uses a scalable modular approach. Simply edit the source files within the <code>src/</code> directory and compile them together using the NodeJS auto-builder
</p>

<pre><code>node build.js
</code></pre>
